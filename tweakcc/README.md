# tweakcc themes

Custom Claude Code themes for [tweakcc](https://github.com/Piebald-AI/tweakcc).

| Theme | File |
|-------|------|
| Ayu Dark | `ayu-dark.json` |
| Gruvbox Dark | `gruvbox-dark.json` |
| Solarized Dark | `solarized-dark.json` |

## Apply

1. `npx tweakcc` → **Themes** → create/edit a theme, copy the `rgb(...)` values from the JSON.
2. Main menu → **Apply customizations** (or `npx tweakcc --apply`).
3. Run `claude`, then `/theme` to select the theme.

Files use the [claude-code-themes](https://github.com/Piebald-AI/claude-code-themes) community format.
